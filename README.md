# intervencion-comunitaria-almargen
Este repositorio documenta el trabajo de cooperación técnica realizado para el Grupo de Compras Comunitarias "Al Margen" en San Carlos de Bariloche. El proyecto fue desarrollado en el marco de la materia Ciencia, Ética, Intervención Socioambiental y Socioproductiva (Optativa) de la UNRN (2024).
