# Intervención Técnica: Optimización del Carrito de Compras Web - Al Margen

Este repositorio documenta el análisis y la propuesta de mejora técnica para la **plataforma de compras virtual** del Grupo de Compras Comunitarias "Al Margen" en Bariloche. Proyecto desarrollado para la materia **Ciencia, Ética, Intervención Socioambiental y Socioproductiva** (UNRN, 2024).

## Contexto del Proyecto
El "Carrito de Compras" es la interfaz digital que permite a los vecinos y organizaciones realizar pedidos de alimentos de forma colectiva. La intervención se centró en mejorar la **usabilidad y eficiencia** de esta herramienta tecnológica para fortalecer la economía popular local.

## Diagnóstico de la Plataforma Web
A través de un diagnóstico participativo con los usuarios (nodos y consumidores), se identificaron necesidades críticas en el sistema:
- **Mejora en la Interfaz de Usuario (UI):** Necesidad de un diseño más intuitivo para facilitar la carga de pedidos.
- **Optimización de Procesos:** Reducción de pasos innecesarios en el flujo de compra para evitar errores en la carga de datos.
- **Accesibilidad:** Adaptación técnica para que la web funcione correctamente en diversos dispositivos y conexiones de red.

## Propuesta de Intervención
El trabajo propone una serie de mejoras técnicas basadas en principios de **Tecnología Social**:
1. **Rediseño del Flujo de Usuario:** Simplificación del proceso de selección de productos y cierre de carrito.
2. **Arquitectura de Información:** Reorganización de las categorías de productos para una búsqueda más ágil.
3. **Sostenibilidad Técnica:** Recomendaciones para el mantenimiento y escalabilidad de la página web utilizando herramientas de código abierto.

## Ética y Tecnología
Este proyecto demuestra cómo la ingeniería de software puede aplicarse con una perspectiva social, priorizando la **ética profesional** y el apoyo a organizaciones territoriales que promueven el comercio justo.
